# Pelisellin puheohjaus

Projektissa hyödynnetään paikallista tekoälymallia ääniavustajan toteuttamiseen. Avustajan päätehtävänä olisi hallita mm. 
Home Assistant järjestelmän valoja. 

## Backend

Backend on tarkoitus toteuttaa käyttäen Ollamaa, LangChainia ja FastAPIa.

## Frontend

Frontend on tarkoitus toteuttaa Reactilla, jossa olisi tarkoitus käyttää jotakin puheentunnistus palvelua.
